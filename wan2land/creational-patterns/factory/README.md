# 팩토리 (Factory)

생성패턴의 가장 기본이 되는 팩토리 패턴입니다. 어떤 객체를 생성할 때 사용합니다.

사용자를 만들어야 할 때, 보통은 클래스 인스턴스를 바로 생성합니다만, 팩토리패턴을 사용하면 다음과 같이 작성할 수 있습니다.

```text
┌───────────────┐
│  UserFactory  │
├───────────────┤
│+ factory()    │
└───────────────┘
        │        
        ▼        
┌───────────────┐
│     User      │
└───────────────┘
```
